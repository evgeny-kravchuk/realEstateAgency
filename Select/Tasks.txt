Задания к л/р №3:

1.	Получить список договоров аренды, срок которых истекает на дату выполнения запроса.
2.	Получить общий список риэлтеров и клиентов, телефонный номер которых содержит ‘+38050’.
3.	Получить список клиентов, владеющих недвижимостью во всех районах.
4.	Подсчитать суммарную стоимость объектов каждого типа, сопровождаемых каждым риэлтером.
5.	Для объектов конкретного района (на усмотрение студента) уменьшить цену на 10%.
6.	Получить список типов объектов, ранжированных по количеству клиентов, когда-либо их арендовавших.
7.	Удалить из БД конкретного клиента (на усмотрение студента). При необходимости выполнить несколько запросов.
8.	Получить список районов, ранжированных по популярности.
9.	Для всех объектов конкретного адреса (на усмотрение студента) установить состояние «под снос».
10.	Получить список объектов (тип объекта на усмотрение студента), удовлетворяющих пожеланиям клиента.
